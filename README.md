# CleanHTTP

Human like HTTP Library powered by fpclient and tls_client.

## Install
```
go get -u github.com/Implex-ltd/cleanhttp/cleanhttp
```

## Issues

```
$env:GOPRIVATE="github.com/Implex-ltd/"
$env:GOSUMDB="off"

```
